# Workflow

## System Workflow

```text
Student
    │
    ▼
Google Form
    │
    ▼
Google Sheet
    │
    ▼
Apps Script Trigger
    │
    ▼
Generate Leave Letter
    │
    ▼
Send Email
    │
    ▼
Student Receives Email
```

## Description

1. Student fills the Google Form.
2. Google Form stores responses in Google Sheets.
3. Apps Script is triggered automatically.
4. The script reads the submitted data.
5. A professional leave letter is generated.
6. Gmail sends the leave letter to the student's email.
